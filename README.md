# Flutter Chat App

Una aplicación de chat en tiempo real construida con Flutter y Firebase.

## Características
- ✅ Chats privados entre usuarios
- ✅ Sistema de presencia en tiempo real  
- ✅ Notificaciones push
- ✅ Interfaz moderna y elegante

## Tecnologías
- Flutter
- Firebase Auth
- Cloud Firestore
- FCM (Notificaciones)
